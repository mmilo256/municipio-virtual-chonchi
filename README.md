# Municipio Virtual Chonchi

El proyecto **Municipio Virtual Chonchi** es una plataforma web que permite a los ciudadanos realizar solicitudes a la municipalidad de Chonchi de forma en línea, con un sistema de seguimiento de solicitudes. Además, incluye un panel de administración para que los funcionarios de la municipalidad gestionen estas solicitudes y realicen las acciones necesarias.

## Estructura del Proyecto

Este repositorio contiene dos aplicaciones principales:

### 1. Portal Web Público
- **Descripción**: Es la aplicación web donde los ciudadanos pueden acceder mediante su Clave Única, realizar solicitudes y hacer un seguimiento de las mismas.
- **Características**:
  - Registro de usuarios mediante Clave Única.
  - Creación de solicitudes de servicios municipales.
  - Seguimiento en tiempo real del estado de las solicitudes.

### 2. Panel de Administración
- **Descripción**: Es la aplicación utilizada por los funcionarios autorizados para gestionar y administrar las solicitudes realizadas por los ciudadanos.
- **Características**:
  - Gestión de solicitudes realizadas por los ciudadanos.
  - Modificación de estados de las solicitudes.
  - Envío de notificaciones a los ciudadanos.
